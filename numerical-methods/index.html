Numerical Methods with Pharo

Pharo is a clean, innovative, open-source, live-programming environment.

Numerical Methods with Pharo covers many topics related to data science: function interpolation, iteration, zeroes, linear algebra series, statistics, estimation, minimisation, and data mining.
